# Project 3: Data Aggregation & Performance Metrics
**DecodeLabs Data Analytics Internship**

## Overview
Analyzed product sales distributions and transaction metrics across 1,200 e-commerce orders to evaluate category volume and unit value trends.

## Key Findings
- Total sales volume led heavily by Chairs with 534 units sold
- Tablets command the highest average product value at $371.06 per unit
- Phones represent the lowest fulfillment volume with only 367 units sold
- High-volume consistency across core categories (Printers 508, Laptops 501)

## Method
Data extraction and connection handled via Power Query using blank queries to process the relational logic. Data aggregation, transactional cross-tabulation, and descriptive summary statistics were then structured into automated performance sheets using Pivot Tables to isolate volume and valuation metrics.

## Tools Used
Microsoft Excel (Power Query, Pivot Tables)
