## Project 3: SQL Data Analysis
**DecodeLabs Data Analytics Internship**

## Overview
Extracted, grouped, and aggregated backend e-commerce data containing 1,200 transaction records using relational database querying and structured queries.

## Key Findings
- Total transaction metrics reveal Chairs led volume (562 units sold) while Phones had the highest average pricing tier ($375.22).
- Isolated exactly 250 operational exception records to group cancellation metrics across all product inventories.
- Total product revenue calculations show Phones generated the highest baseline turnover ($54,211.88) while Monitors ranked the lowest ($38,021.74).
- Order fulfillment status breakdowns indicate "Delivered" accounts for the majority of the transactional pipeline.

## Method
Relational database structuring and data extraction utilizing multi-layer SELECT filters, WHERE constraint funnels, GROUP BY categorical buckets, and aggregate functions (SUM, AVG, COUNT) inside an SQL database engine. Results compiled into clean tabular layouts.

## Tools Used
SQLite (SQLite Online), Microsoft Excel
